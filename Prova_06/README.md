# [JSIA-A06] Gerenciador de Lista de Tarefas em JavaScript

Este projeto consiste em um programa para gerenciar e processar uma lista de tarefas utilizando funções anônimas, arrow functions e funções de callback.

## Requisitos do Projeto

- **Funções Anônimas:**  
    - Uma função anônima atribuída a uma variável é usada para adicionar tarefas ao array de tarefas.

- **Arrow Functions:**  
    - Uma arrow function lista todas as tarefas, mostrando cada tarefa com seu índice no console.

- **Funções de Callback:**  
    - Uma função recebe outra função como callback, permitindo executar operações como:
        - Remover uma tarefa.
        - Atualizar uma tarefa.
        - Concluir uma tarefa.

- **Interação com o Usuário:**  
    - O programa utiliza `prompt()` para receber ações do usuário e exibe os resultados no console com `console.log()`.

## Exemplo de Uso

1. O usuário pode adicionar tarefas à lista.
2. Listar todas as tarefas cadastradas.
3. Remover, atualizar ou concluir tarefas usando funções de callback.
4. Todas as interações são feitas via prompt e exibidas no console.

## Tecnologias Utilizadas

- JavaScript (ES6+)
- Métodos de interação: `prompt()`, `console.log()`
